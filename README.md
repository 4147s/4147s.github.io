A specialized tool designed to assist users in managing and navigating YouTube channel status and feature accessibility settings. 

**Live Link:** [4147s.github.io/features?nv=1](https://4147s.github.io/features?nv=1)

---

## Features

* **Feature Management:** Quickly access hidden or deep-linked YouTube feature settings.
* **Status Check:** Review the standing and monetization eligibility of your channel.
* **Responsive Interface:** Fully optimized for both desktop and mobile browser environments.
* **One-Click Navigation:** Streamlined URL parameters to jump straight to specific verification pages.

## Getting Started

### Prerequisites
To use this tool effectively, you should be logged into your **Google/YouTube account** in the same browser session.

### Usage
1. Visit the hosted site: `https://4147s.github.io/features?nv=1`
2. Navigate through the dashboard to select the specific YouTube feature you wish to review.
3. Follow the on-screen prompts to update your channel status.

## Local Development

If you want to run this project locally or contribute to the code:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/4147s/4147s.github.io.git
    ```
2.  **Navigate to the directory:**
    ```bash
    cd 4147s.github.io
    ```
3.  **Open the project:**
    Simply open `index.html` in your preferred browser.

## Configuration
The tool uses the `nv=1` parameter to initialize specific navigation states. You can modify the URL parameters to test different routing behaviors within the app.
